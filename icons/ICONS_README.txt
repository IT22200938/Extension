ICON FILES NEEDED
=================

This extension requires icon files in the following sizes:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using any image editor or online tool.

RECOMMENDED TOOLS:
1. Online Icon Generators:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
   
2. Design Tools:
   - Figma (free online)
   - Canva (free online)
   - GIMP (free desktop app)

DESIGN SUGGESTIONS:
- Use a simple, recognizable symbol (e.g., tracking target, analytics chart, cursor icon)
- Use the extension's color scheme: purple gradient (#667eea to #764ba2)
- Ensure the icon is visible on both light and dark backgrounds
- Keep the design simple for smaller sizes (16x16 and 32x32)

TEMPORARY SOLUTION:
Until you create proper icons, you can:
1. Use free icon sets from:
   - https://www.flaticon.com/
   - https://iconmonstr.com/
   - https://www.iconfinder.com/
   
2. Or create simple colored squares as placeholders using online tools

NOTE: The extension will still work without custom icons, but browsers may show a default placeholder icon instead.
